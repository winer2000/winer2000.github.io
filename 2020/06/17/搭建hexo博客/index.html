hexo博客搭建

Windows平台

准备工作

​ 1）安装Nodejs

​ 直接点击下一步下一步就行了

​ 2）安装Git

​ 和Nodejs安装大概相同都只需点击下一步等待安装完成就好了

​ 安装完成之后需要添加github账号和邮箱

​ 添加/修改用户名 git config –global user.name

​ 添加/修改用户邮箱 git config –global user.email

​ 检查当前账号 git config user.name

​ 检查当前邮箱 git config user.email

​ 3）检查安装情况及版本

​ 在Git中输入node -v 查看Nodejs是否安装成功并查看其版本

​ npm -v查看npm是否安装成功并查看其版本

搭建过程

​ 1）将npm挂载到国内的镜像源(建议更换，速度会比原来要快)

​ 查看当前npm的镜像网站 npm get registry

​ 更换当前的npm镜像网站 npm config set registry URL(镜像网站的地址,国内推荐阿里云)

​ 安装淘宝cnpm管理器 npm -g install cnpm –registry=http://registry.npm.taobao.org

​ 查看cnpm的版本 cnpm -v

​ 2）安装hexo框架

​ 安装hexo框架 cnpm install -g hexo-cli

​ 查看hexo框架版本 hexo -v

​ 3）正式创建博客

​ 在自己指定的目录下创建一个blog目录

​ 利用cd命进入到blog目录

​ 在blog目录生成hexo博客(初始化hexo博客) sudo hexo init

​ 启动本地博客服务 hexo s

​ hexo博客本地访问地址 http://localhost:4000/

​ 创建新的文章 hexo n “文章标题”

​ 返回到blog目录

​ 清理hexo博客中产生的垃圾文件 hexo clean

部署到GitHub仓库

​ 1）在GitHub上新建一个新的仓库YourGithubName.github.io(注意名字一定不能错)

​ 2）配置blog目录中的_config.yml文件

​ deploy:

​ type: git

​ repo:https://github.com/YourGithubName/YourGithubName.github.io.git

​ branch: master

​ 3）生成部署文件 hexo g

​ 4）清理博客中多余的文件 hexo c

​ 4）将hexo 博客部署到GitHub仓库中 hexo d

​ 5）在浏览器中输入https://YourGithubName.github.io访问你的hexo博客
